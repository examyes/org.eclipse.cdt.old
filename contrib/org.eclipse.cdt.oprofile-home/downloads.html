Installation

When installing Eclipse-OProfile from a .tar.gz archive, you must do the following:

  1. Remove any previous Eclipse-OProfile installations.
  2. Copy the eclipse-oprofile.tar.gz archive to the directory above your Eclipse installation directory. For example, an ls in this directory should show your eclipse installation directory.
  3. Change into the directory mentioned above and type: tar zxf eclipse-oprofile.tar.gz
  4. From the same directory, type: cd eclipse/plugins/org.eclipse.cdt.oprofile.core.linux*
  5. Set the executable bits on the install scripts by typing: chmod +x install && chmod +x update-perms
  6. Install authentication-related files (you must have root privileges to do this) by typing: ./install
  7. Update permissions on Eclipse-OProfile executables by typing: ./update-perms
