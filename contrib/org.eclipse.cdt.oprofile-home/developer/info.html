Eclipse-OProfile Developer Information

Developer Information and How to Get Involved
CVS Repository

The Eclipse-OProfile sources are available under anonymous CVS access:

$ cvs -d :pserver:anonymous@dev.eclipse.org:/home/tools login

Press 'enter' when prompted for a password.

$ cvs -d :pserver:anonymous@dev.eclipse.org:/home/tools co -z9 \
	org.eclipse.cdt-contrib/org.eclipse.cdt.oprofile
Developer Communication

All communication related to Eclipse-OProfile development should be directed toward the cdt-contrib-dev@eclipse.org mailing list.

Patches

We love patches! If you have a patch to submit, first open a Bugzilla ticket, then attach your patch to the open bug. A committer will review your patch and commit it to the source repository upon approval. Alternatively, you can post your patch to the cdt-contrib-dev@eclipse.org mailing list.

We use GNU-style ChangeLog entries to track changes made to source files. When you post a patch, you generally paste your ChangeLog entry above the beginning of the patch in the patch file.

Building Eclipse-OProfile

We use the standard Eclipse PDE build procedure. To build Eclipse-OProfile, you must have a suitable platform driver and CDT build installed. Then, execute the following procedure on your build machine:

  1. Check out the Eclipse-OProfile releng plug-in: 
    $ cvs -d :pserver:anonymous@dev.eclipse.org/home/tools co -z9 \
	org.eclipse.cdt-contrib/org.eclipse.cdt.oprofile/org.eclipse.cdt.oprofile.releng
  2. Start the build: 
    $ cd /path/to/org.eclipse.cdt.oprofile.releng
$ export ECLIPSE_HOME=/path/to/eclipse/dir
$ java -cp $ECLIPSE_HOME/startup.jar \
	org.eclipse.core.launcher.Main \
	-application org.eclipse.ant.core.antRunner \
	-DbaseLocation=$ECLIPSE_HOME \
	-Dpde.build.scripts=$ECLIPSE_HOME/plugins/org.eclipse.pde.build_3.1.0/scripts \
	-Dconfigs=linux,gtk,x86
    NOTE: If you are building on an architecture other than x86, modify the last line accordingly. The build configuration must match an assembly target in the customTargets.xml file. If your desired configuration does not have an assembly target, write one and send us a patch.
  3. As usual, the PDE builder will deposit a distributable archive (.tar.gz) for the build in the results directory.
Branch Information

All development for Eclipse-OProfile with OProfile 0.5.4 is done on oprofile-0_5_4-branch. This branch is mostly in maintenance mode. The mainline (HEAD) is currently active for Eclipse-OProfile development with OProfile 0.8.x.
