About Eclipse-OProfile

Eclipse-OProfile is a graphical, Eclipse-based front-end to the powerful OProfile profiling tool. Eclipse-OProfile is integrated with the C/C++ Development Tools (CDT) and aims to provide users with an easy-to-use interface for profiling C/C++ applications.

Interested parties are strongly encouraged to participate in the development of Eclipse-OProfile. Get involved in Eclipse-OProfile!

The OProfile back-end is required by Eclipse-OProfile and is only available on the Linux operating system. The OProfile back-end is a standalone, separately-maintained open source project and has no affiliation with Eclipse-OProfile.

Eclipse-OProfile is licensed under the terms of the Eclipse Public License and is implemented in Java as a set of plug-ins to the Eclipse platform. One small exception is the opxml application, which is licensed separately under the terms of the GNU General Public License.

News

Eclipse-OProfile is now available!

Red Hat is pleased to announce the immediate availability of Eclipse-OProfile on Eclipse.org! Red Hat has donated Eclipse-OProfile to the Eclipse Foundation's C/C++ Development Tools project and is actively encouraging other participants to get involved in development.

Until the new Eclipse.org mailing list is set up, please use the cdt-dev@eclipse.org mailing list for all developer communication and please send patches to the cdt-patch@eclipse.org mailing list.

Also, please use Red Hat's Bugzilla until the new Eclipse.org Bugzilla components are set up. Eclipse-OProfile bugs should be filed under the Red Hat Developer Suite product, version 2.0, Profiling Plug-in.
