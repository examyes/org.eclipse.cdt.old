Viewing executable profiling statistics

Executables are displayed when the session name is "expanded" in the System Profile view. (To expand a session, click on the small triangle to the left of its name.)

To view sample counts for a given executable, in the System Profile view:

  1. Select the event.
  2. Select the session.
  3. Select the executable.

The Sample view displays:

  • The start VMA (address) of the symbol in which the sample was collected.
  • The total Count of samples in the symbol.
  • The percentage (%age) of the total counts of this executable collected in the symbol.
  • The Name of the symbol.

If no symbol information is found in the executable, then the System Profile view displays the actual samples collected. Similarly, the Sample view displays information about the samples.
