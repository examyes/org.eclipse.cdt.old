Viewing sample information

To view the samples collected in given symbol (if symbol information exists in an executable), expand the desired symbol in the System Profile view. A list of samples appears. Select a sample; the Sample view shows the following information about the sample:

  • The VMA (address) of the sample.
  • The percentage (%age) of the total counts of this executable collected in this sample's address.
  • The Line number in the source file of the sample's address (only if there is debugging information in the executable).
  • The source file Name in which this sample's address occurs (only if there is debugging information in the executable).

For any executable with debugging information (that is, Line and Name are not blank), double-click the sample in the Sample view to open an editor for the source file and position the cursor at the given line number. If the source file is in a CDT project, the project's editor will open the file; otherwise a generic editor opens the file.
