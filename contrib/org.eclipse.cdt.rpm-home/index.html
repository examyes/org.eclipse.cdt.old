About Eclipse-RPM

Eclipse-RPM is a graphical, Eclipse-based tool for packaging applications in the popular RPM Package Manager (RPM) format. Eclipse-RPM works alongside the C/C++ Development Tools (CDT) and aims to provide users with an easy way to package C/C++ applications in the RPM format.

Interested parties are strongly encouraged to participate in the development of Eclipse-RPM. Get involved in Eclipse-RPM!

The RPM Package Manager, which is required by Eclipse-RPM, is available for many different architectures and operating systems. The RPM Package Manager is a standalone, separately-maintained open source project and has no affiliation with Eclipse-RPM.

Eclipse-RPM is licensed under the terms of the Eclipse Public License and is implemented in Java as a set of plug-ins to the Eclipse platform.

Eclipse-RPM is now available!

Red Hat is pleased to announce the immediate availability of Eclipse-RPM on Eclipse.org! Red Hat has donated Eclipse-RPM to the Eclipse Foundation's C/C++ Development Tools project and is actively encouraging other participants to get involved in development.

Please use the new cdt-contrib-dev@eclipse.org mailing list for all developer communication. For more information about how to get involved in Eclipse-RPM development, have a look at the developer information.
