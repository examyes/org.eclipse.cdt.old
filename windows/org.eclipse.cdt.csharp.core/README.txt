This plug-in requires LPG V1.1. It can be found from the following site:

   //http://sourceforge.net/project/showfiles.php?group_id=155963&package_id=190492

Download the lpgdistribution-05-16-06.zip file and extract it somewhere. Import the
lgpdistribution folder as an "Existing Project into Workspace". It is a Java project
but also includes the lpg executable used by the parser generator external tool,
CSharpParser.launch.
