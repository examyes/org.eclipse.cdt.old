Installation - Plugin

To install the C/C++ Development Plugin:

  1. Ensure you have installed the prerequisites
  2. Go to the Websphere Studio Workbench plugins directory ( <...>\eclipse\plugins\ )
  3. Install the C/C++ Development Plugin 
     tar zxvf C_IDE_Linux_R1.0_mmddyy.tgz
