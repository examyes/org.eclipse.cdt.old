text parameter

The text parameter may be used to query or set the current element's text.

Availability

query command
set command

Scope

The current element of the current view.

Syntax

query text
set text elementText

Parameters

elementText Use the elementText parameter to specify the text for the current element.

Examples

query text
set text This is some text for the current element.
