formatLineText parameter

The formatLineText parameter may be used to query or set the text that is displayed in the format line.

Availability

query command
set command

Scope

The current view.

Syntax

query formatLineText
set formatLineText [ text ]

Parameters

text Use the text parameter to specify the text that should be displayed in the format line. If you do not specify any text, the format line will display the column positions in the editing area.

Examples

query formatLineText
set formatLineText ----+----1----+----2----+----3//COMMENTS
set formatLineText


query command
set command
current.formatLine parameter
