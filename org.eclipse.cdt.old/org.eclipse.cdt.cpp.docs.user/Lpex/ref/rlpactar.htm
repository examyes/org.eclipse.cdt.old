actionArgument parameter

The actionArgument parameter may be used to set or query the argument that will be used by the next editor action.  The interpretation of the argument is specific to the action.

Availability

query command
set command

Scope

The current view.

Syntax

query actionArgument
set actionArgument argument

Parameters

argument Use the argument parameter to specify the desired action argument.

Examples

query actionArgument
set actionArgument 4


query command
set command
actionRepeat parameter
appendToActionArgument action
setActionArgument action
