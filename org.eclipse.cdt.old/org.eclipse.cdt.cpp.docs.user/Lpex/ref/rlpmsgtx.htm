messageText parameter

The messageText parameter may be used to query or set the text that is displayed in the message line.

Availability

query command
set command

Scope

The current view.

Syntax

query messageText 
set messageText text

Parameters

text Use the text parameter to specify the text that should be displayed in the message line.

Examples

query messageText 
set messageText Good afternoon!


query command
set command
