parse command

The parse command can be used to trigger the parser.

Syntax

parse [ all ]

Parameters

all Use the optional parameter all to indicate that the all of the lines in the document should be parsed. If this option is not specified, only the lines that are currently on the parse pending list will be parsed.

Status

The status parameter is not affected by this command.

Examples

parse 
parse all
