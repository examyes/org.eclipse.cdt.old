prefixText parameter

The prefixText parameter may be used to query or set the current element's prefix text.  The prefix area is the area to the left of the text area that is visible when current.prefixArea is on.  Prefix text is normally entered by the user, and is used by the processPrefix command to run prefix commands for base profiles like ispf, seu, and xedit.

Availability

query command
set command

Scope

The current element of the current view.

Syntax

query prefixText
set prefixText text

Parameters

text Use the text parameter to specify the prefix text for the current element.

Examples

query prefixText
set prefixText cc
