block.type parameter

The block.type parameter may be used to query the current block selection type.  The block selection type may be one of stream, character, element, or rectangle.  If there is no block selection, block.type will return none.

Availability

query command

Scope

Global.

Syntax

query block.type

Examples

query block.type
