markId parameter

The markId parameter can be used to query the id of the specified mark.  The id is a positive integer value which uniquely identifies a mark.  This parameter is usually used to determine the id of an unnamed mark immediately after it has been created.

Availability

query command

Scope

The specified mark in the current view.

Syntax

query markId.[name | #id]

Description

If you do not specify a mark name or a mark id, the last mark created will be used.

Examples

query markId.
query markId.test
query markId.#3
