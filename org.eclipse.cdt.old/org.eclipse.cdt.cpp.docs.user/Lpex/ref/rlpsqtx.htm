sequenceText parameter

The sequenceText parameter may be used to query or set the current element's text part of the sequence numbers.

Availability

query command
set command

Scope

The current element of the current view.

Syntax

query sequenceText
set sequenceText text

Parameters

text Use the text parameter to specify the element's sequence-numbers text part.

Examples

query sequenceText
set sequenceText 991231
