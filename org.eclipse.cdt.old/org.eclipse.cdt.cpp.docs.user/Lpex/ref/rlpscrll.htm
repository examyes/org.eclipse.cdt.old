scroll parameter

The scroll parameter may be used to query or set the number of pixels that the current view is scrolled to the right.

Availability

query command
set command

Scope

The current view.

Syntax

query scroll
set scroll n

Parameters

n Use the n parameter to indicate the number of pixels that you want the current view to be scrolled to the right.

Description

Note that the scroll value that you indicate will be adjusted so that the cursor remains visible.

Examples

query scroll
set scroll 20
