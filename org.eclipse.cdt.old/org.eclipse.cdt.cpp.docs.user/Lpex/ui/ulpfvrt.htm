Application Preferences Dialog - Named_file

Use the controls in this preference group to give the selected file a new name and/or location.

Controls and push buttons are:

Name Initially contains the file name and path location of the file. You can specify a new file and directory location by entering the new information in this field. The file is not actually renamed until you save it.
Push buttons
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
