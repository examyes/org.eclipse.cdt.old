Application Preferences Dialog - Save (for named file)

Use the controls in this preference group to specify file save settings for a specific file.

Controls and push buttons are:

Trim trailing blanks If selected, the editor trims off extra blanks from the end of each line before saving the file.
Maximum line length Enter an integer value to specify a maximum length for each line. Characters beyond the specified length are truncated and lost when the file ids savedl is.

If a value of 0 is specified, line length is limited only by system restrictions.
Push buttons
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
