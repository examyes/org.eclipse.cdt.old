Application Preferences Dialog - Parser (for named file)

Use the controls in this preference group to specify the parser to be used for the specified file.

Controls and push buttons are:

Parser Lists parsers known to the editor. Select a parser from the list to apply that parser to the selected file.
Push buttons
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
