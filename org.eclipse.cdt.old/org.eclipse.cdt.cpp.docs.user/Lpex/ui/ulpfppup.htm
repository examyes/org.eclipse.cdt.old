Application Preferences Dialog - Popup menu

Use the controls in this preference group to add or modify entries in Editor pane pop-up menu.

Controls and push buttons are:

Current list of pop-up menu items Displays a list of current Editor pane pop-up menu entries. Click on any entry in this list to see the characteristics and/or actions associated with that entry.
Text Displays the text label associated with an existing menu entry. You can modify a text label for an existing menu entry or enter new text for a new menu item or submenu.
Action Displays the Editor action or user-defined action associated with an existing menu entry. You can modify an action for an existing menu entry or enter a new action for a new menu item..
Radio buttons The type of menu entry selected is indicated by the active radio button. You cannot change the type of an existing menu entry, but you can specify a type of your choice for a new menu entry.
Menu item
You must specify a Text name and Action for the new menu item.
Submenu
You can only specify a Text name for a submenu entry. Once a submenu is created, you can create new menu items to go with that submenu.
Separator
This specifies a separator bar in your pop-up menu. No Text name or Action can be specified with this type of menu entry.
Push buttons
Set
Saves the settings specified by the above controls to the selected pop-up menu entry.
Delete
Deletes the selected menu entry from the pop-up menu.
Add after
Saves the settings specified by the above controls to a new pop-up menu entry that will be inserted after the currently-selected menu item or separator.
Add as child
Saves the settings specified by the above controls to a new pop-up menu entry that will be added as a submenu item to the currently-selected submenu.
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
