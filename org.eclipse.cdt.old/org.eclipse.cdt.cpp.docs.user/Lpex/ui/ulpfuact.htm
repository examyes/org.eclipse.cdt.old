Application Preferences Dialog - User actions

Use the controls in this preference group to associate an action name with an action class name.

Controls and push buttons are:

Current action name/action class association table This table shows current action name and action class associations defined to the editor. To modify an existing entry in this table, click on any part of an entry's line. The current values of the table entry will be placed in the Name and Class name fields where they can be edited as required.
Name Enter an action name to which you want to associate an action class, or edit an existing action name in this field.
Class name Enter a name of an action class to which you want to associate an action name, or edit an existing action class name in this field.
Push buttons
Set
Saves a new or modified user action class association.
Delete
Deletes the currently-selected user action class association.
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
