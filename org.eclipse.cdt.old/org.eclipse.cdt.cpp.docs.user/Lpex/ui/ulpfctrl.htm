Application Preferences Dialog - Controls

Use the checkbox controls in this preference group to specify what controls and information areas are displayed in the editor. Checkbox controls and push buttons are:

Status line If selected, displays a status line below the Editor pane title bar showing:
  • Cursor row and column location
  • Insert/replace mode
  • Number of changes to file since last save
Format line If selected, displays a format line area below the status line in the Editor pane.
Message line If selected, displays a message line area below the editing area in the Editor pane.
Command line If selected, displays an editor command entry area at the bottom of the Editor pane.
Line numbers If selected, line numbers are displayed in the editing area. These numbers are not saved as part of the file.
Hide sequence numbers If selected, sequence numbers are hidden. This option has effect only for those files that already have had sequence number areas defined in the Applications Preferences Sequence numbers or File open panes.
Expand/hide area If selected, a symbol is shown whenever a filter or find operation causes a group of lines not matching filter or find criteria to be hidden from view. Clicking on the symbol brings the hidden lines, preceded by a symbol, back into view. Clicking on the symbol hides those lines again.
Push buttons
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
