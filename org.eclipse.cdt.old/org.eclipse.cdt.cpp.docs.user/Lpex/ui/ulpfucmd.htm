Application Preferences Dialog - User commands

Use the controls in this preference group to associate a command name with an command class name.

Controls and push buttons are:

Current command name/command class association table This table shows current command name and command class associations defined to the editor. To modify an existing entry in this table, click on any part of an entry's line. The current values of the table entry will be placed in the Name and Class name fields where they can be edited as required.
Name Enter a command name to which you want to associate a command class, or edit an existing command name in this field.
Class name Enter a name of a command class to which you want to associate a command name, or edit an existing command class name in this field.
Push buttons
Set
Saves a new or modified user command class association.
Delete
Deletes the currently-selected user command class association.
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
