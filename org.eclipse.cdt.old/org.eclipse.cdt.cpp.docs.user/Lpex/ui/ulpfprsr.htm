Application Preferences Dialog - Parsers

Use the controls in this preference group to associate a parser class with a parser name.

Controls and push buttons are:

Current parser name/parser class association table This table shows current parser name and parser class associations defined to the editor. To modify an existing entry in this table, click on any part of an entry's line. The current values of the table entry will be placed in the Name and Class name fields where they can be edited as required.
Name Enter a parser name to which you want to associate a parser class, or edit an existing parser name in this field.
Class name Enter a name of a parser class to which you want to associate a parser name, or edit an existing parser class name in this field.
Push buttons
Set
Saves a new or modified parser class association.
Delete
Deletes the currently-selected parser class association.
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
