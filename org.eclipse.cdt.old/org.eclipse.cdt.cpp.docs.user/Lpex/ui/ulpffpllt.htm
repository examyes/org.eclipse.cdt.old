Application Preferences Dialog - Color palette

Use the controls in this preference group to specify the palette used in the Editor pane.

Controls and push buttons are:

Color palette Lists color palettes available to the editor. Colors in this list denote the background color of the palette.
Push buttons
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
