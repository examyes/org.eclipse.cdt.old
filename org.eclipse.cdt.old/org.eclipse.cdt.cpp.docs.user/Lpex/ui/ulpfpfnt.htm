Application Preferences Dialog - Print font

Use the controls in this preference group to specify the print font used when printing files from the editor. As you make selections, the Sample text string in the center of this pane changes to show the effect of your current selections.

Controls and push buttons are:

Use the screen font Instructs the editor to use the native screen font when printing files. If this option is selected, the Name, Style, and Size selections are disabled.
Name Lists fonts available to the editor for printing.
Style Select any of the available entries in this group to add special text effects to printed text.
Size Enter an integer to increase or decrease the size of the font used when printing text.
Push buttons
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
