Application Preferences Dialog - Print

Use the controls in this preference group to specify the page and content options used when printing files from the editor.

Controls and push buttons are:

Line numbers If selected, line numbers are added before each line when a file is printed from the editor.
Tokenized If selected, highlighting effects used by the parser to display language elements are sent to the printer.
Top margin Enter an integer to increase or decrease the size of the top margin when printing text.
Bottom margin Enter an integer to increase or decrease the size of the bottom margin when printing text.
Left margin Enter an integer to increase or decrease the size of the left margin when printing text.
Right margin Enter an integer to increase or decrease the size of the right margin when printing text.
Push buttons
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
