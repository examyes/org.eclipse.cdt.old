View Pulldown Menu

Actions and selections in the View pulldown menu let you define how the various parts of the editor and information in those parts is viewed. Depending on the context of the specific editor window you are working in, selections in the View pulldown menu will include some or all of the following:

Refresh file tree Updates the directory and file tree information shown in the File Selector pane. This action is disabled when showing information for open or recently-opened files.
Refresh drive list Updates the disk drive and directory information shown in the drive selection list at the top of the File Selector pane. This action is disabled when showing information for open or recently-opened files.
Remove directory from drive list Removes the entry for the directory path displayed in the File Selector pane from the drive selection list. This action is enabled only for directory paths previously added to the drive selection list.
Reload file Reloads the current file from disk. All changes made since the last save of the file are lost.
Restore pane If the currently-selected pane has previously been minimized or maximized, this action restores that pane to its previous size.
Maximize pane Maximizes the width of the currently-selected pane so it occupies all of the editor window.
Minimize pane Minimizes the width of the currently-selected pane until only the restore button remains visible.
Detach pane Detaches the currently-selected pane from the current editor window. The detached pane is placed in its own editor window together with only the pulldown menus and menu items that pertain to the contents of the now detached pane. The pane remaining in the original editor window, if one exists, expands to fill the entire space of that editor window and the contents of the pulldown menus change to suit the context of the remaining pane.
Return to the parent page Appears only in the View pulldown menu of detached panes. Moves the contents of the current editor window back to the editing window from which those contents were originally detached.
File Selector If checked, the left side of current editing window displays the File Selector pane. Use this pane to select or open files already open in the editor, files recently opened, or any other file stored on your system.
Editor If checked, the left side of current editing window displays the Editor pane. Use this pane to view and edit the contents of files.
