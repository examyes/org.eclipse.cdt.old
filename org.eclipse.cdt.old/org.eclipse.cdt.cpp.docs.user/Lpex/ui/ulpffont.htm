Application Preferences Dialog - Font

Use the controls in this preference group to specify the font used in the Editor pane. As you make selections, the Sample text string in the center of this pane changes to show the effect of your current selections.

Controls and push buttons are:

Name Lists fonts available to the editor.
Style Select any of the entries in this group to add special text effects to text displayed in the Editor pane.
Size Enter an integer to increase or decrease the size of the font used in the Editor pane.
Push buttons
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
