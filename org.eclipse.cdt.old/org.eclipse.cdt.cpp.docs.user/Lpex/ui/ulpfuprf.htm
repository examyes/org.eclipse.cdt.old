Application Preferences Dialog - User profile

Use the controls in this preference group to apply a user profile class to the current editing session.

Controls and push buttons are:

User profile Displays the name of the user profile class currently in effect. To change the profile, type the name of the replacement profile into this field.
Push buttons
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
