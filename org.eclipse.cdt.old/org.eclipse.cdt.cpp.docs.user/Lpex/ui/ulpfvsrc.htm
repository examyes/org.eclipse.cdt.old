Application Preferences Dialog - Source encoding (for named file)

Use the controls in this preference group to specify how the editor opens and stores the contents of a specific file. Controls are:

Source encoding The editor can work with files originating from or intended to be targeted towards a platform, such as an S/390 host system, other than that of the workstation on which the editor is running. This group of controls lets you specify the character encoding used to represent the file being loaded into the editor.

The current value in the Source encoding field is initially the name of the native character encoding set used by the workstation on which the editor is running. You can change this to another character encoding set by double-clicking on the choices provided in the selection list below the Source encoding field, or by directly typing in the name of the character encoding set.
Push buttons
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
