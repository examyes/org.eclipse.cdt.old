Application Preferences Dialog - Compare

Use the checkbox controls in this preference group to specify comparison criteria used when comparing files with the Edit->Compare->Compare... action. Checkbox controls and push buttons are:

Ignore case If selected, differences in character case are ignored for comparison purposes.
Ignore leading blanks If selected, leading blanks at the start of a line are ignored for comparison purposes.
Ignore trailing blanks If selected, trailing blanks at the end of a line are ignored for comparison purposes.
Ignore all blanks If selected, all blanks are ignored for comparison purposes.
Push buttons
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
