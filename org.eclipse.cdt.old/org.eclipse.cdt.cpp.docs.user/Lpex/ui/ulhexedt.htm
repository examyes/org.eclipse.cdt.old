Hex Edit Line Dialog

This dialog lets you edit a line of text using any combination of:

  • Ordinary text.
  • Unicode character representation.
  • Native character representation used by the workstation on which the editor is being used.
  • Source character representation as used by a file originating from or targeted towards a host system, with a character encoding representation different from that of the workstation on which the editor is being used.

The dialog contains the following controls and fields:

Text Contains the plain text representation of the current line.
Unicode Contains the Unicode representation of the current line.
Native encoding Contains the native encoding representation of the current line.
Source encoding Contains the source encoding representation of the current line. This line appears only if a source encoding mode is selected for the current file.
Push buttons
OK
Applies current selections to the editor and closes the Hex Edit Line dialog.
Cancel
Cancels current selections and closes the Hex Edit Line dialog.
Help
Displays this help.
