Editor Main Menubar

Each entry on the main menubar gives you access to a pulldown menus containing groups of related actions.

The default set of main menubar items and actions available from their corresponding pulldown menus are:

File Gives access to basic file operations, such as:
  • File open and close actions
  • Editor application preference selection
  • Editor close actions
Edit Gives access to editing options relating to text in the current editing window, such as:
  • Undo and redo actions
  • Cut, copy and paste actions
  • Find and replace actions
  • Text filtering
  • Location marking
  • Text comparisons with other files
  • Keystroke recording and playback
View Gives access to viewing options relating to the current editing window, such as:
  • Pane minimization and maximization
  • Detached panes
  • Active pane selection
Selected Gives access to actions pertaining to files or text in the active editor pane.

If the File Selector pane is active, this pulldown menu lets you work with entries in the pane, such as:

  • Open files selected in the File Selector pane
  • Manage fast-access file and directory entries in the File Selector pane

If the Editor pane is active, this pulldown menu lets you perform editing-related actions, such as:

  • Cut, copy, and paste actions
  • File insert, save and print actions
  • Undo and redo actions
  • File location marking
  • Text and location find operations
  • Filter operations
  • Keystroke recording
  • Application and editor preference selection
Window Gives access to window-related actions, such as:
  • Opening new editing windows apart from the current editing window
  • Selecting the current editing window from a list of open editing windows
Help Gives access to online help and editor documentation selections.
