Application Preferences Dialog - User key actions

Use the controls in this preference group to associate a file name extension with a parser name.

Controls and push buttons are:

Current user key actions table This table shows current key and action associations defined to the editor. To modify an existing entry in this table, click on any part of an entry's line. The current values of the table entry will be placed in the Key and Action fields where they can be edited as required.
Key Displays the name of a key to which you want to associate an action. For example, c-t is the name for the Ctrl+T key combination. You can modify the key name for an existing menu entry or enter a new key name.
Action Displays the editor action or user-defined action associated with an existing menu entry. You can modify an action for an existing key or enter a new action for a new key.
Push buttons
Set
Saves a new or modified user key action.
Delete
Deletes the currently-selected user key action.
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
