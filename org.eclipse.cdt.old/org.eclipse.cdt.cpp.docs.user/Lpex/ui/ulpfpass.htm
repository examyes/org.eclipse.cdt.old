Application Preferences Dialog - Parser associations

Use the controls in this preference group to associate a file name extension with a parser name.

Controls and push buttons are:

Current file extension/parser association table This table shows current file extension and parser name associations defined to the editor. To modify an existing entry in this table, click on any part of an entry's line. The current values of the table entry will be placed in the File extension and Parser fields where they can be edited as required.
File extension Enter a new file extension to which you want to associate a parser name, or edit an existing file extension in this field.
Parser Enter a name of a parser to which you want to associate a file extension, or edit an existing parser name in this field.
Push buttons
Set
Saves a new or modified parser association.
Delete
Deletes the currently-selected parser association.
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
