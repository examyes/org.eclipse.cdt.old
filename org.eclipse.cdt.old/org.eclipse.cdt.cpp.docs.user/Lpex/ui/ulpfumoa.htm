Application Preferences Dialog - User mouse actions

Use the controls in this preference group to associate a file name extension with a parser name.

Controls and push buttons are:

Current user mouse actions table This table shows current mouse event and action associations defined to the editor. To modify an existing entry in this table, click on any part of an entry's line. The current values of the table entry will be placed in the Mouse event and Action fields where they can be edited as required.
Mouse event Displays the name of a mouse event to which you want to associate an action. For example if you want to specify an action for double clicking mouse button 1 with the Ctrl and Alt keys pressed, you can specify c-a-1-clicked.2.You can modify the mouse event name for an existing menu entry or enter a new mouse event name.
Action Displays the editor action or user-defined action associated with an existing mouse event entry. You can modify an action for an existing mouse event or enter a new action for a new mouse event.
Push buttons
Set
Saves a new or modified user key action.
Delete
Deletes the currently-selected user key action.
Apply
Applies current selections to the editor and leaves the Application Preferences dialog open.
Reset
Cancels current selections and leaves the Application Preferences dialog open.
Default
Restores default editor settings to options in this pane.
