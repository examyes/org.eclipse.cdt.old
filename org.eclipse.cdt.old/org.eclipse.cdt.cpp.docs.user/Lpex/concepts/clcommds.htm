Editor commands, actions, and parameters

The editor is fully programmable through the use of an extensive command, action, and parameter set.

You can use commands and parameters to customize your editor window, search for or change text in your document, or perform many other functions. Editor actions are usually associated with menus, keys, mouse actions, and prefix commands, and may be used for programming the editor.

You can find more information about using commands, actions, and parameters in the related readings below.
