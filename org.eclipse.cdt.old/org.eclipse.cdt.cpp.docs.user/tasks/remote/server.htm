Running the Server/Daemon

Server: Running the Server/Daemon

Server: Installation

  • If you downloaded the full version of the CppPlugin, then make sure that the following directories exist in your plugins directory.

    /home/user/eclipse/plugins/com.ibm.dstore.extra.server
    /home/user/eclipse/plugins/com.ibm.dstore.core
    /home/user/eclipse/plugins/com.ibm.dstore.miners
    /home/user/eclipse/plugins/com.ibm.cpp.miners
    /home/user/eclipse/plugins/com.ibm.cpp.miners.parser
    /home/user/eclipse/plugins/com.ibm.debug
    /home/user/eclipse/plugins/com.ibm.debug.gdbPicl

  • If you only downloaded the server version then:
    1. choose a directory on your server machine to install the DataStore server 
      /home/user/dstore/
    2. copy the C_IDE_Linux_Server_R1.0_mmddyy.tgz file to the server machine in the chosen location
    3. After you unzip the server package, you should have the following directories:
      /home/user/dstore/com.ibm.dstore.extra.server
      /home/user/dstore/com.ibm.dstore.core
      /home/user/dstore/com.ibm.dstore.miners
      /home/user/dstore/com.ibm.cpp.miners
      /home/user/dstore/com.ibm.cpp.miners.parser
      /home/user/dstore/com.ibm.debug
      /home/user/dstore/com.ibm.debug.gdbPicl

Running a DSTORE SERVER

  • You need to run the script "server.linux" which is located in the directory com.ibm.dstore.core. So, if you have the full version of the CppPlugin: 
    	cd /home/user/eclipse/plugins/com.ibm.dstore.core
	./server.linux
  • If you only have the server version then: 
    	 cd /home/user/dstore/com.ibm.dstore.core
	 ./server.linux

Running a DSTORE DAEMON

  • Make sure that you are running as root. If the daemon is not run under root, it will be unable to authenticate connecting users.


  • You need to run the script "daemon.linux" which is located in the directory com.ibm.dstore.core. So, if you have the full version the CppPlugin: 
    	cd /home/user/eclipse/plugins/com.ibm.dstore.core
	./daemon.linux
  • If you only have the server version (i.e. downloaded "C_IDE_Linux_Server_R1.0_mmddyy.tgz") then: 
    	 cd /home/user/dstore/com.ibm.dstore.core
	 ./daemon.linux

Running a DStore Server

  1. Change directory to 
    /home/user/eclipse/plugins/com.ibm.dstore.core
  2. Type the following to start a DStore Server:
    java -DA_PLUGIN_PATH=$A_PLUGIN_PATH com.ibm.dstore.core.server.Server [port]

    NOTE: By default the port is 4033.

Running a DStore Daemon

  1. Make sure that you are running as root. If the daemon is not run under root, it will be unable to authenticate connecting users.


  2. Type the following to start a DStore Daemon:
    java -DA_PLUGIN_PATH=$A_PLUGIN_PATH com.ibm.dstore.core.server.ServerLauncher
