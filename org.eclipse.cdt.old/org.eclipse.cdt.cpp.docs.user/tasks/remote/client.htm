Running the Client

You start the client by creating a project with the Project Wizard and specifying information for the Host machine(i.e. the machine where the server/daemon is running). Pick any port number above 4000 if the remote machine is running the daemon.



If you do not know the exact location of your source files in the remote machine, then you can browse its filesystem from the root by typing "/" in the Directory field and clicking the Browse button.
