Launch Commands

The Command View provides a launching point for system commands where parsed output is fed into the Command Output View. To launch a command:

  1. Specify the working directory, by either selecting a directory or file in the Navigator view, or by using the Browse Button in the Command View
  2. Enter the Command, by typing the command and clicking the Run Button. See Troubleshooting commands for an explanation of why pressing <Enter> will not work, and some other limitations.

After the command has been launched you will immediately be taken to the Command Output View to Examine the output or Cancel the Command.
