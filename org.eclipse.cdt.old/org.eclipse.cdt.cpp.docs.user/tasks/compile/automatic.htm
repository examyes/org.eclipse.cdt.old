Using the Automatic Build Feature

By default the Automatic Build Feature is enabled. This setting controls whether a Project-level Build will be performed whenever you change a resource (e.g. save a file, create a new file, delete a file, etc.).

This setting is accessed from the Workbench menu:


After toggling this setting to off, you will be able to invoke builds manually, when deemed desirable.
