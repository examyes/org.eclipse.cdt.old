Autoconf Tutorial

Managing Project Using Autoconf Support

The following is a tutorial that shows how to use CDT support for the GNU autoconf and automake tools to manage a newly created project. We will use a  simple project - HelloWorld App-  in this tutorial as the focus here is to explain the various feature that CDT provides.

1- Creating C++ project
 

  • The first step is to successfully create a new cpp project and set its location to be the default location.
  • Create a subdir which will include the source file(s) for your C++ Program:
    1. From the C/C++ project view, RMC(right mouse click) the project and select "New -> Directory"
    2. Enter the name of the directory and press OK
  • Create a new cpp file by RMC the directory that you have created and select New - > File, then enter tha file name HelloWorld.C
  • Now we need to write the HelloWorld.C:


    • You can simply copy the following code to your file in the Eclipse workspace or type the following:
     

      //include this file for cout
      #include <iostream.h>

      int main() 
      {  

         //print out the text string, "Hello, World!"
         cout << "Hello,  World!" << endl;  

         return 0; 
      }


    Before we move to the next part make sure that you have the following project structure:

    ProjectFolder

      Source Folder
        Source Files
