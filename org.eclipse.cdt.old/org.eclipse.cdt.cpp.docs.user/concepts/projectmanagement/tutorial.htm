Autoconf Tutorial

Managing Project Using Autoconf Support

The following is a tutorial that shows how to use CDT support for the GNU autoconf and automake tools to manage a newly created project. We will use a  simple project - HelloWorld App-  in this tutorial as the focus here is to explain the various feature that CDT provides.

1- Creating C++ project
 

  • The first step is to successfully create a new cpp project and set its location to be the default location.  You may give it the name HelloApp.
  • Create a subdir which will include the source files for your Application:
    1. From the C/C++ project view, RMC(right mouse click) the project and select "new -> directory"
    2. Enter the name of the directory and press OK
