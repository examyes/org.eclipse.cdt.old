Drop And Dragging Views

Although this is a feature of the Eclipse Workbench, we though it appropriate to discuss this here.

To further tailor your work environment, you can drop and drag views within your workspace. When you drag a window you can either drop it

  1. in a different window grouping
  2. create a new window outside existing window groups
  3. place it on the left-hand toolbar.

When you drop a window on an existing window group it is always added on the right-hand side of the windows list.

Different Window Grouping


Simply click and drag the view tab to the other window group and when the cursor turns into a group of folders


release the mouse button.
Outside Window Groups


Simply click and drag the view tab to the other window group and when the cursor turns into an arrow


release the mouse button.
Left-Hand Toolbar


Simply click and drag the view tab to the other window group and when the cursor turns into a group of folders


release the mouse button.


Selecting the icon will show the window


selecting the icon a second time will hide the window.

Selecting the icon will restore the window to it's original location or window group.
