C/C++ Projects View

The C/C++ Projects View provides a means of navigating through and working with the files of your project(local or remote). Many of the actions presented are specific to the C/C++ Plugin.

Files

For a file to appear in the C/C++ Projects view, it must be part of a project. See Create a Project for details on creating projects.

Once your source files are in the C/C++ Projects view, you may navigate and perform actions on them.
