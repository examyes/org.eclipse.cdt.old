Targets View

The Targets View is used to maintain a list of targets for each project which exists in the Navigator. Targets for each project are displayed in a table tree fashion from where the user can easily create, edit and build targets. The view flattens out the targets and views them in the same level regardless of their location in the project hierarchy. Targets are the nodes of the tree while the leaves are the properties of each target. The user can edit and update these properties, as well as perform actions on the selected target directly from the view. The complete list of actions will be described below in the Targets view features section.

Creating New Targets

There are three ways to start the Target SmartGuide.

  1. From the Navigator view:
    • By right mouse clicking on the project in the Navigator view, and selecting menu item Create Target from the pop up context menu

  2. From the Targets view:
    • By right mouse clicking the view background, and selecting menu item New Target from the pop up context menu


    • From the view tool bar

Target SmartGuide

The Target SmartGuide will be shown after initiating the Create Target action. The current project will be highlighted automatically, indicating the current target container. Users can change the container and select the desired sub folder. Next, input the target name in the Target name text field and the target invocation command in the invocation field.

Click

Click to expand the target to see the associated attributes.

Limitations
  1. The maximum number of targets per project is 100
