Creating your own custom perspective

There are two ways to create your own custom perspective:

  1. Open the C/C++ Perspective and add or close views as required
  2. Start with the default perspective and select the views you require.

You can also choose to:

  1. Modify your existing workbench
  2. Open a new workbench.
Modifying the C/C++ Perspective
