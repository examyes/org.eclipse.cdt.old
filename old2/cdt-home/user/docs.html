Introduction

The C/C++ development User Guide is a work in progress that is available in the form of online help from the Eclipse Workbench. Updates to the User Guide will be committed to CVS on a weekly basis.

Opening the C/C++ Development User Guide

To open the CDT online help

  1. Click Help > Help Contents > C/C++ Development User Guide.
  2. Click on the Contents tab in the lower left corner of the Help - Eclipse Platform window.
  3. Click on C/C++ Development User Guide.

Contributing Feedback

Any comments or queries regarding the C/C++ Development User Guide may be directed to https://dev.eclipse.org/mailman/listinfo/cdt-doc-dev. Feedback on the help (to the cdt-doc-dev list), is strongly encouraged. Comments on the scope, content, clarity, usability are particularly sought.

Also, send a message to the list if you want to contribute to the online help or volunteer to be a reviewer.
