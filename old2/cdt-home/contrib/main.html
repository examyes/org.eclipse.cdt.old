Example Plugins

  • GNU Indent, formatter
    • Unpack the zip-file
    • make 'indent' available on your system:
      1. under Linux go to http://www.gnu.org/software/indent/indent.html#TOCdownloading download the sources and compile it.
      2. under Windows you can try the same, but it is much easier to use cygwin (you will find indent in the 'devel'-section. To make it work, make sure that your cygwin-bin-folder is in your PATH.
    • Enable in the preference page; Preferences -> C/C++ -> Code Formater, choose in the drop menu "GNU Indent".
    • Use ctl-shift-F or the the context menu to reformat selection or the entire file.
    • See the GNU indent man to see how to set the options.
  • Generic Binary Parser
    • Enable in the property Page, select "Generic Binary Parser".
  • GDB Idle Launcher

ChangeLog Plugins

This plugin allows the user to press a key sequence while in a module and the ChangeLog file for that module will pop up in an editor with the developer's name and e-mail address (previously entered via a preference page) and the date. The plugin uses the built-in parsers in Eclipse and currently supports Java/C/C++.
