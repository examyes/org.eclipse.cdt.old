Release Builds Update Site

CDT release builds are made available on the CDT release update site. Use the following URL in a Site Bookmark in the update manager:

http://update.eclipse.org/tools/cdt/updates/release

To access the Update Manager, select Software Updates->Update Manager from the Help menu. To access an update site, select New->Site Bookmark from the context menu in the Feature Updates view. In the New Site Bookmark dialog, enter a name for the site (any name) and enter the URL for the update site. You can then expand the site bookmark in the Feature Updates view to reveal the available downloads.

Currently, CDT releases 1.2.1, 1.2 and 1.1 are available as well as releases of CDT/CPPUnit Integration. As new releases are made and new features are added, they will appear in this update site.

For those who need the flexibility of the old zip distributions, they are available here.

CDT 1.2 Note: For CDT 1.2, the standard make project type and the new managed make project type have been separated out from the CDT "Platform".  When using the update site, please install the CDT feature as well as the Standard Build feature and optionally the Managed Make feature, in that order, or optionally selecting them and installing them at the same time (Process All). Other release notes for CDT 1.2 are available here.

Even more important note on CDT 1.x: CDT versions 1.x are note likely to run with Eclipse 3.0 series builds. Your best best is to stay on the 2.1.x series. There is an Eclipse effort to support binary compatibility with plugins built against 2.1.x. As soon as those efforts have achieved success, this note will be deleted.

CDT 2.0 Development Stream

The CDT 2.0 stream is currently under development. The objective of this stream is to keep up with the Eclipse 3.0 development and is targeted at release in sync with Eclipse 3.0, i.e. June 2004.

For those who would like to try the latest bits, the nightly builds are available on the CDT 2.0 builds update site. Be warned, however that these bits have not been thoroughly tested:

http://update.eclipse.org/tools/cdt/updates/builds/2.0

CDT 2.0 M7

The CDT 2.0 M7 build is available from the following update site. This build will only run against Eclipse 3.0 M6 and later. Please follow this link to find the install directions.

http://update.eclipse.org/tools/cdt/updates/builds/2.0M7

CDT 2.0 M8

The CDT 2.0 M8 build is available from the following update site below. This build (or later) is required for users that wish to use Eclipse 3.0 M8.

        http://update.eclipse.org/tools/cdt/updates/builds/2.0M8

CDT 2.0 M9

The CDT 2.0 M8 build is available from the following update site below. This build (or later) is required for users that wish to use Eclipse 3.0 M9. This is the last milestone build prior to reaching General Availability (GA) for the 2.0 release of the CDT.

    http://update.eclipse.org/tools/cdt/updates/builds/2.0M9
