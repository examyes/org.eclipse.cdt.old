Latest CDT Downloads 
On this page you can find the latest builds of the C/C++ Development Tools (CDT).

To use these plugins, you must first install the compatible version of the Eclipse platform as indicated in the table below.   You may download Eclipse from the Eclipse project download page.
For Eclipse Release 1.0

These plugins require Eclipse Release 1.0 for Linux Motif.
Aside from Debug Support, Autoconf, and Linux Help Integration, the CDT will run on Eclipse 1.0 for Windows NT or 2000, but we do not claim to support Windows at this time. (Note: If you are going to use the CDT on Windows, we highly recommend using Windows NT or 2000, since on Windows 98 it has been reported that running commands does not currently work)
Stable Builds are done from the CVS head (working towards CDT R1.0).
