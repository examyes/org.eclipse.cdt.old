Nightly Builds 

CDT Nightly builds have not been tested at all and are provided "as is". It is possible that these builds have major bugs and may not run at all. The only check that is performed by the script that generates this page is a quick glance at the size of the Error\Warning log from the build. In the table below the glyphs indicate the following:
 
- indicates the the Error\Warning log is not any larger than expected, so there is a good chance that there were no serious errors. 
- indicates the the Error\Warning log is larger than expected which probably means the build is broken.
