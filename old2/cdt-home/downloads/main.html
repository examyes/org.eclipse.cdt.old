Downloads For Eclipse Release 2.0
These plugins require a pre-release version of Eclipse Release 2.0 for Linux Motif. Since Eclipse R2.0 drivers are being updated regularly, we can not guarantee that these drivers will work with any other (later) versions than noted.
Aside from Debug Support, Autoconf, and Linux Help Integration, the CDT should run on Eclipse R2.0 drivers for Windows NT or 2000, but we do not claim to support Windows at this time. (Note: If you are going to use the CDT on Windows, we highly recommend using Windows NT or 2000).
Stable Builds are done from the HEAD in CVS.
