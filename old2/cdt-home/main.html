About the CDT Subproject

The CDT project provides a set of plugins that implement a C/C++ IDE. It adds a C/C++ Perspective to the Eclipse Workbench that supports C/C++ development with a number of views, wizards, a powerful editor, autoconf support, remote project support and a debugger. More details...

Note: the CDT is primarily focused on Linux Application Development. Some functionality such as the debugger and autoconf support may not work on other clients such as Windows. However, we are interested in the participation of others to extend the CDT .
Plans & Schedules
