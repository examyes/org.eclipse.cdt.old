Resources

The CDT project provides a set of plugins that implement a C/C++ IDE. It adds a C/C++ Perspective to the Eclipse Workbench that supports C/C++ development with a number of views, wizards, a powerful editor, and a debugger. The CDT is designed to provide an extensible architecture, that will provide support for integration of tools provided by Independent Software Vendors.

CDT Components

The Overview of the CDT Project can be viewed here

The CDT subproject is broken down into components. Each component operates as an autonomous project, with its own set of committers, bug categories and mailing lists. Each component supports information regarding its requirements, design, planning/milestones and test plans.

CDT Components Description
Primary CDT Plugin Eclipse Primary CDT Plugin
CDT Feature Eclipse CDT Feature Component
CDT Core Core Model, CDOM and Core Components
CDT UI Core UI, Views, Editors and Wizards
CDT Launch Generic Launch Mechanism
CDT Debug Core Core Debug Model
CDT Debug UI Core Debug UI, Views, Editors and Wizards
CDT Debug MI Debug controller that interfaces to MI-capable debuggers
