C/C++ Development Tools

The CDT project provides a set of plugins that implement a C/C++ IDE. It adds a C/C++ Perspective to the Eclipse Workbench that supports C/C++ development with a number of views, wizards, a powerful editor, autoconf support, remote project support and a debugger. For more general information see CDT Overview and Project Plans or take a look at some Screenshots.

Note: the CDT is primarily focused on Linux Application Development. Some functionality such as the debugger and autoconf support may not work on other clients such as Windows. However, we are interested in the participation of others to extend the CDT

CDT Components

The CDT subproject is broken down into components. Each component operates as a autonomous project, with its own set of committers, bug categories and mailing lists.
