Importing Existing Documents
How do you bring an existing project into Eclipse/CDT so that you can work on it from within the CDT.
Author : Doug Schaefer
Revision Date : 09/04/2003 - Version: 0.1.0
Change History : 0.1.0 - Document Creation

A common scenerio that users run into when starting to use the CDT is how to start using it with existing source code. Eclipse supports a number of ways of doing this. The following are recommended approaches to doing this for the CDT.

Create New Projects for Existing Source Roots

This method is best used when the source for the project exists on the file system accessible to the user's desktop.  The idea is to create a new CDT project that is rooted on each root directory that contains the source.
