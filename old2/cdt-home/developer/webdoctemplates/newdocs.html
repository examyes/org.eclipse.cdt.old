1. Create the web pages

In an attempt to unify the appearance of the CDT webpages, we now have a Cascading Style Sheet called cdt.css. It is recommended that you use the CDT Web Template as a starting point. If however you wish to create a webpage from scratch, then this style sheet can be included in your HTML documents by adding the following the HEAD section:

  <LINK rel="stylesheet" type="text/css" 
   HREF="http://dev.eclipse.org/viewcvs/index.cgi/%7Echeckout%7E/cdt-home/cdt.css?cvsroot=Tools_Project">

The CDT Style Sheet gives us a central location to specify the look of CDT web pages. This is accomplished by defining how various HTML entities should be displayed. Below is the current list of styles and how they appear:
