Project Structure

The CDT project itself is organized into autonomous components each with its own set of committers, bug categories and mailing lists.

As of Feb 4/2004 the CDT project is broken down as follows:

The CDT subproject is broken down into components. Each component operates as an autonomous project, with its own set of committers, bug categories and mailing lists. Each component supports information regarding its requirements, design, planning/milestones and test plans.
