  1. >How do I develop (build\run\debug) CDT?

    The CDT can be built from source either inside Eclipse or from the command line. In addition, Eclipse provides powerful Java Development (Edit, Compile, Debug) facilities that make developing the CDT plug-ins very easy. Please see Developing the CDT for more information.
