1. Introduction

1.1 Prerequisites

The CDT is written in java and depends on many parts of the Eclipse base, so the following must be on your system before building the CDT:

  1. JDK version 1.3 or greater
    • Find a JDK for your system here
  2. A version of Eclipse that matches the level of CDT you intend to check out of cvs
    • Find out what Eclipse version you need here
  3. Linux\Unix System Tools
    • The makefile-based build process uses the following Linux\Unix System Tools:
      - make, find, test, cat, rm, xargs, and others
      These tools should be standard on most Linux\Unix systems. On Windows these tools can be obtained by installing Cygwin
OPTIONAL
  1. JavaCC - only required if you intend to make changes to and regenerate the C/C++ Parser)
    • Get the latest version of JavaCC here
