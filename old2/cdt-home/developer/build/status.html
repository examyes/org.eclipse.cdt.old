CDT Versions in CVS

When checking the CDT out of CVS you can optionally include a tagname that checks out versions of the CDT source that build with particular Eclipse builds. So for example, if you were working with Eclipse R1.0 you would check out CDT using the following command:

cvs checkout -r R1-0 cdt
