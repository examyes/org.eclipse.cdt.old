Eclipse
Embedded Systems Birds of a Feather Eclipse Event (April 2003)
BOF participants

24 April 2003, San Francisco

An Eclipse BOF (with an emphasis on the embedded industry) was held at the ESC show in San Francisco. Attendance to the BOF was quite good, with about 30 attendees from the embedded tools industry present.

Presenters

Chris Laffra, IBM. Chris gave an overview of the Eclipse platform and Java tooling. He also showed a really cool tool which he developed in his spare time, which shows the activity and interaction between the different plugins in Eclipse. Chris' powerpoint slides can be found here.

Sebastien Marineau, QNX. Sebastien gave an overview of the CDT project, including the CDT's architecture, features as well as roadmap. Sebastien's powerpoint slides can be found here.

Cory Bialowas, Rational/IBM. Cory discussed Rational's contributions to the CDT project, which was followed by a demo of CDT 1.1, including the new code parser. Cory's powerpoint slides can be found here.

Manas Saksena, Timesys. Manas presented Timesys's Timestorm IDE, and highlighted the areas where Timesys did specific integration with their build and debug environment. Manas followed this by a live demonstration of Timestorm.

Chris Songer, Tensilica. Chris presented some of the work Tensilica has been doing with Eclipse. In particular, Chris demoed performance and optimization tools which have been prototyped at Tenslica. Chris' powerpoint slides can be found here.
